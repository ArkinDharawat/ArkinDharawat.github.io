Here are some of my side projects, class projects and 3am-ideas

Image Similarity using Deep Ranking

Re-implementation of paper, "Learning Fine-grained Image Similarity with Deep Ranking". Trained ResNet101 neural net on TinyImageNet across multiple GPUS. Achieved an outstanding 73.6% top-5 test accuracy.

Web of Videos

A Web app that connects video segments across various online courses. The segments are connected by the similarity of concepts covered in them. I implemented and experimented with various text analysis algorithms such as Word2Vec, semantic similarity and tf-idf scoring to make a similarity metric.

Scientific Paper Recommendation System

This was a project to develop a recommendation system for scientific papers using databases and unsupervised topic models. We used Neo4J and SQL to store and model the data and Gensim and spaCy to parse and build the LDA topic model.

FlickrHIV-Image/Image-Meta dataset

Created image and image-metadata dataset as a part of SocialMedia Heatlh research group. Implemented mining infrastructure in Python and made use of mining and analysis techniques to measure tag co-occurrence,query expansion and filtering noisy or incorrect tags.

Slack Summarizer

A Slack-bot that summarizes chats in various channels. Implemented a modified Text-rank with weights taken from concept extraction for summarization Numpy for matrix multiplication and NetworkX for graphs and the famous Pagerank Algorithm
